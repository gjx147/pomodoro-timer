# 番茄钟桌面应用

一个简洁高效的番茄工作法计时器，支持番茄模式和游戏模式。

![番茄钟](https://img.shields.io/badge/Pomodoro-Timer-4CAF50?style=flat-square)

## 功能特点

### 🎯 番茄模式
- 25分钟专注工作
- 5分钟短休息
- 每4个番茄后15分钟长休息
- 番茄计数显示

### 🎮 游戏模式
- 40分钟游戏计时
- 10分钟休息
- 按次计数

### ⚙️ 其他功能
- 桌面通知提醒
- 声音提醒
- 主题切换（浅色/深色）
- 窗口置顶
- 最小化到系统托盘
- 右键托盘菜单（显示/退出）

## 安装运行

### 开发模式
```bash
npm install
npm start
```

### 构建 exe
```bash
npm run build
```

构建完成后，exe 文件位于 `dist/win-unpacked/` 目录。

## 技术栈

- **框架**: Electron
- **前端**: HTML5 + CSS3 + JavaScript
- **构建工具**: electron-builder

## 使用说明

1. **开始计时**: 点击"开始"按钮
2. **暂停**: 点击"暂停"按钮暂停计时
3. **重置**: 点击"重置"回到当前模式初始时间
4. **跳过**: 点击"跳过"直接进入下一个阶段
5. **切换模式**: 点击"番茄模式"或"游戏模式"切换
6. **主题切换**: 点击右上角设置按钮，选择主题
