# 番茄钟桌面应用规范

## 1. 项目概述

- **项目名称**: Pomodoro Timer
- **类型**: 桌面应用程序 (Electron)
- **核心功能**: 番茄工作法计时器 + 游戏模式，帮助用户专注工作或游戏
- **目标用户**: 需要时间管理的上班族、学生

## 2. 功能列表

### 番茄模式
- 25分钟专注工作计时
- 5分钟短休息计时
- 15分钟长休息计时（每4个番茄后触发）
- 开始/暂停/重置/跳过控制
- 番茄计数显示（4个点）

### 游戏模式
- 40分钟游戏计时
- 10分钟休息计时
- 按次计数
- 开始/暂停/重置/跳过控制

### 通知功能
- 计时结束时的桌面通知
- 计时结束时的声音提醒
- 托盘图标右键菜单（显示/退出）

### 界面功能
- 模式切换（番茄模式/游戏模式）
- 最小化到系统托盘
- 窗口置顶选项
- 主题切换（浅色/深色）

## 3. 视觉规范

### 布局
- 单窗口设计，居中显示
- 顶部：应用标题（powered by 郭建巡）+ 设置按钮
- 模式选择按钮
- 中部：圆形计时器显示
- 底部：控制按钮 + 番茄计数

### 配色
- 主色（绿色）: #4CAF50
- 灰色: #808080（深）/ #E0E0E0（浅）
- 浅色主题背景: #F5F5F5
- 深色主题背景: #2D2D2D

### 字体
- 主字体: "Segoe UI", system-ui, sans-serif
- 计时器数字: "SF Mono", Consolas, monospace, 72px

## 4. 技术方案

- **框架**: Electron
- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **构建**: electron-builder
- **窗口**: 400x500px，不可调整大小

- [ ] 番茄模式计时器准确运行（25分钟工作/5分钟短休/15分钟长休）

- [ ] 模式切换功能正常
- [ ] 暂停后继续计时正确
- [ ] 重置恢复到当前模式初始时间

- [ ] 桌面通知正常弹出
- [ ] 声音提醒正常播放
- [ ] 可最小化到托盘
- [ ] 托盘菜单功能正常
